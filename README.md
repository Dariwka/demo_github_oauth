# demo_github_oauth 
## DEMO login with github and showing data from github API

## Getting Started

### Dependencies

Cookie-session, cors, dotenv, express, nodemon, passport, passport-github2

### How to start
cd api, npm start (localhost:5000)
cd client, npm start (localhost:3000)
